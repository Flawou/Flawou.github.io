Remote Desktop Services (RDS)

Le RDS est un composant de Microsoft Windows qui permet à un utilisateur d'accéder à des applications et des données sur un ordinateur distant, via n'importe quel type de réseau.

Fonctionne sur le port TCP RDP 3389

protocole RDP ( Remote Desktop Procotol)

 

Il a pour avantages :

  1. l'accès à distance sans restriction
  2. Gestion simplifié (gestion applications et mise à jour centralisées)
  3. centralisation ressource ( réduire la charge de travail sur poste client)
  4. économie de coûts ( si serveur puissant, réduit les coûts des postes clients, license)
  5. Multi-users ( Prends en charge plusieurs sessions simultanées 
  6. Compatibilité (évite les problèmes de compatibilité entre différentes versions d'applications, car les applications sont exécutées sur le serveur)
  7. sécurité  (donnée et applications restent sur le serveur, pratique en cas de vol ou perte matériel)

Inconvénient : 

  1. Dépendance à la connectivité réseau / bande passante réseau
  2. Coûts initiaux de la mise en place ( serveur / license / configuration )
  3. Configuration complexe 
  4. Sécurité compromise en cas de mauvaise configuration
