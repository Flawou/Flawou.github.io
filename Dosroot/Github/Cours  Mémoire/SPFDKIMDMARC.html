SPF/DKIM/DMARC

SPF = Sender Policy Framework

moyen pour un domaine de répertorier tous les serveurs à partir desquels il envoie des e-mails

DKIM = Domain Keys Identified Mail

Signature via clé privé/publique

DMARC = Domain-based Message Authentication Reporting and Conformance

vérifie SPF et/ou DKIM et décide de comment agir en fonction de ses résultats
