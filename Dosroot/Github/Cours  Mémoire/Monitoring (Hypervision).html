Monitoring (Hypervision)

Le monitoring ou monitorage est une activité de surveillance et de mesure d'une activité informatique. On l'emploie pour permettre la supervision de son parc informatique. 

Il est possible de le mettre en place avec différents logiciels tel que : 

Zabbix / Centreon / Nagios Core / Icinga etc

Il est nécessaire de l'utiliser pour surveiller l'état de son parc informatique 

(consommation ressource / gestion ressource )

état hardware / état services 

Permet d'être pro-actif (prévenir les pannes) 

snmp

simple network management protocol
