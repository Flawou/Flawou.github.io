Rejoindre DC W

vérification adressage ip (même réseau que DC) ip -c a ou vérif via interface graphique puis apt upgrade

apt install realmd

realm discover tssr.local

si fonctionne pas > nano /etc/resolv.conf

mettre adresse serveur

installer package affiché par la commande :

apt -y install realmd libnss-sss libpam-sss sssd sssd-tools adcli samba-common-bin oddjob oddjob-mkhomedir packagekit

ensuite joindre DC avec la commande  :

realm join ‐U administrateur tssr.local

realm list pour vérifier la configuration 

 

Si vous voulez rename machine : nano /etc/hostname

config sans interface graphique :

nano /etc/network/interfaces

#the primary netwok interface (ens33 ou ethx)

auto ens33

allow-hotplug ens33

iface ens33 inet static

address 192.168.244.80

netmask 255.255.255.0

gateway 192.168.244.2

dns-nameservers 192.168.244.5 #(DC)

nano /etc/resolv.conf

domain name tssr.local

search domain

mettre adresse serveur
