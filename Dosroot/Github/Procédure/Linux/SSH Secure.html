SSH Secure

sur server

apt install openssh-server

systemctl start ssh

systemctl status ssh

sur client 

ssh -V

ssh user@ip

Pour créer nos clés de sécurités SSH : 

ssh-keygen (NON CONNECTER SSH)

ssh-copy-id user@192.168.X.X (NON CONNECTER SSH)

changer le port d'écoute

PermitRootLogin no

nano /etc/ssh/sshd_config

décommentez Port 22

modifier le port ( exemple 2222)

depuis client pour s'y connecter : ssh user@192.168.217.10 -p 2222

allowUsers user1
