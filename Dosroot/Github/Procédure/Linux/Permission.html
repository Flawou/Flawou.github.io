Permission

chgrp = modifier le groupe du fichier ( changer de groupe ) chgrp dossier groupe

chmod xxx FILE =  gérér droit sur le fichier ( 4 - Read 2 Write 1 Execute )  user/groupe/other

exemple chmod 640 = user peut lire/écrire/execu, groupe peut lire/ecrire et other aucun droit

chown = modification propriétaire fichier
