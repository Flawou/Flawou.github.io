Windows Server

Etape 1 et 2 : Basculer en mise à jour manuel et renommer la machine.

Appuyer simultanément sur la touche windows + R 

taper cmd, enter

taper Sconfig, enter

Taper 5, enter, M, enter 

Taper 2, enter, nommé votre serveur (Win-srv par exemple),enter, relancer la machine

 

Etape 3 : Définir une ip sur la machine

Une fois le démarrage terminé, appuyer de nouveau simultanément sur windows + R

Taper ncpa.cpl

Cliquez droit sur votre interface réseau puis propriétés

Puis sélectionner Protocole internet version 4 (flèche verticale), puis sélectionner propriétés (flèches horiztonale)

 

Une fois l'interface ouverte, cliquer sur Utiliser l'adresse IP suivante, et insérer l'ip voulu.

Le dns est facultatif, à votre bon vouloir.
