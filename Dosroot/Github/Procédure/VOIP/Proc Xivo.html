Proc Xivo

  • installer l'iso de Xivo dans un premier lieu (iso)
  • ensuite récupérer l'ip de la carte réseau (ici c'est 192.168.240.148)
  • dans le navigateur web de sa machine, rentrer l'adresse IP pour accéder au panel de Xivo
  • cliquer sur suivant
  • accepter les termes d'utilisation de Xivo
  • dans la page suivante:
    • Rentrer le nom d'hôte
    • indiquer le domaine (ici c'est tssr.local)
    • mettre un mot de passe administrateur pour accéder au panel (ici c'est root)
    • rentrer 1.1.1.1 comme DNS (ou notre DNS si on en a un)
  • Ensuite:
    • Entrer ce qui servira de nom de serveur (ici c'est XivoSilver)
    • Entrer une intervalle de numéros utilisé pour les numéros internes (ici 100-200)
    • Pareil pour les appels entrants (ici c'est 1000-2000)
  • Je vérifie que ce que j'ai entré est bon puis je valide
  • je me reconnecte a Xivo et il est prêt !
  • A SAVOIR: Pour contacter un mobile, il faut que la VM qui contienne le serveur XIVO soit sur le même réseau que l'appareil:
    • Pour ça, on peut passer la VM en bridge et utiliser le réseau wifi sur lequel le portable est (ou un point d'accès par partage de connexion 4G entre le portable et le serveur)
    • puis sur le terminal du serveur il suffit de faire en sorte que le DHCP prenne en compte le changement de réseau:
      • pour ça il faut marquer dhclient -r
      • puis dhclient -v
      • pour vérifier si c'est bon, faire ip -c a | grep eth0
      • puis on utilise la nouvelle adresse et une application de sotfphone sur portable (comme Zoiper par exemple)
