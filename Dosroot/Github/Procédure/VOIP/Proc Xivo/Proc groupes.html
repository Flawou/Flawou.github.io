Proc groupes

  • Pour pouvoir appeler plusieurs personnes en même temps, il faut les regrouper dans un groupe.
  • Pour ça, Il faut d'abord modifier notre contexte contenant nos rangées de numéros
  • Services --→ IPBX --→ Configuration IPBX --→ Contextes
  • Aller dans les paramètres du contexte default (le crayon a droite de ses infos):
  • Aller dans l'onglet groupes
    • Cliquer sur l'icone “+” verte:
    • indiquer notre intervalle de numéros qui sera utilisé pour les groupes (j'utiliserai 2000-2999):
    • SAUVEGARDER !!!
  • maintenant, aller dans Paramètres IPBX --→ Groupes
  • Cliquer sur l'icone “+”
  • Remplir les cases encadrés:
    • Nom: le nom de votre groupe
    • Numéro: ce qui nous servira a contacter le groupe
    • stratégie de sonnerie: comment vous voulez qu'il appelle les membres du groupe
    • temps de sonnerie: combien de temps il va sonner
    • Délai d'attente pour joindre un utilisateur: combien de temps max pour contacter un membre du groupe
    • Délai d'attente avant rappel des utilisateurs: combien de temps avant qu'il rappel les membres du groupe
    • Contexte: si on a créé un contexte pour ce groupe, on le choisi ici
  • Dans l'onglet utilisateurs, rajouter les membres que vous voulez avoir dedans (ici ça sera Ainashel, Eonder et Alnya), puis on sauvegarde
  • Et voila, maintenant, plus qu'a tester.
