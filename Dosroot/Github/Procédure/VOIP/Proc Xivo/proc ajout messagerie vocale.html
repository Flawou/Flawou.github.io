proc ajout messagerie vocale

  • dans Services --→ IPBX --→ Paramètres IPBX --→ Messageries vocales:
    • cliquer sur l'icone “+” et remplir les cases encadrés:
      • Nom complet: celui de votre utilisateur
      • Numéro de boite vocale: ce qui servira a joindre votre boite vocale (via *99"Numéro de boite vocale) ou *98 si vous appelez votre propre boite.)
      • Langue: fr_FR
      • Joindre le fichier audio: mettre non sauf si vous avez un fichier audio a disposition
      • décocher “Demander le mot de passe”
      • Sauvegarder !!!!
  • maintenant que la boite vocale est crée, il va falloir la lier a l'utilisateur,
  • Pour ça, aller dans Paramètres IPBX --→ Utilisateurs:
  • Cliquer sur le crayon a coté des informations de l'utilisateur a qui on veut lier la boite vocale (ici ça sera Silver Lockser)
  • Aller dans la catégorie Messagerie Vocale,
  • Dans la case rechercher une messagerie, entrer le nom de votre messagerie (ou son numéro si le nom ne marche pas)
  • Sélectionner la messagerie concerné (ici c'est notre Silver Lockser - 501 [default]) et des nouvelles informations s'afficheront,
  • S'assurer que “Activer la messagerie vocale” est cochée puis vérifier les informations avant de sauvegarder
  • maintenant il faut s'assurer que on sera redirigé sur la boite vocale si il n'y a pas de réponse ou si ce numéro est occupé,
  • pour ça, on retourne dans les paramètres de notre utilisateur puis on va dans l'onglet Non réponse
    • Pour “Aucune réponse” sélectionner “messagerie vocale” et sélectionner notre boite vocale
    • Faire pareil pour Occupé et cocher “jouer le message d'occupation”
    • sauvegarder !!
  • NE PAS OUBLIER DE MODIFIER LE PARAMETRE “DTMF” EN RFC 2833 dans la ligne concerné (ici c'est la ligne de Silver Lockser)
