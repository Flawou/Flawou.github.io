proc Contexte

  • Dans Services --→ IPBX --→ Configuration IPBX --→ Contextes, cliquer sur l'icone “+”
  • remplir les cases indiquées:
    • Nom: le nom qu'aura le contexte
    • Nom affiché: le nom qui sera affiché dans les autres paramètres
    • entité: de quel serveur il s'agit
    • type de contexte: quel type d'appels il concerne (ici on va faire un contexte interne)
    • On inclut les contextes qui recevront les appels venant des numéros configuré dans ce contexte (vu qu'on veut que des numéros du contexte Direction puissent appeler et recevoir des appels de numéros venant du contexte Usine, on inclut Usine par exemple)
  • Ensuite, selon ce que l'on veut faire (groupes, utilisateurs, etc…) on sélectionne l'onglet concerné (ici on veut pouvoir faire des utilisateurs  utilisant des numéros dans un intervalle configuré dans ce contexte, donc on va dans l'onglet Utilisateurs puis on rajoute notre intervalle qui sera 6000 - 6500)
  • on sauvegarde !!!
