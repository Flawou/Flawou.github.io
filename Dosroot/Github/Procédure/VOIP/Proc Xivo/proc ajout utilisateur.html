proc ajout utilisateur

  • sur le panel Xivo, je vais dans Services --→ IPBX
  • dans “Paramètres IPBX” --→ Utilisateurs :
    • Cliquer sur l'icone “+” puis sur Ajouter
  • remplir les cases encadrées dans l'image en dessous (Mettre la langue en fr_FR et le fuseau horaire en Europe/Paris) :
  • Ensuite, aller dans l'onglet Lignes et cliquer sur l'icone “+” verte :
  • Remplir les cases de la nouvelle ligne comme tel (pour le numéro ce sera celui qu'on utilisera pour appeler cet utilisateur, donc 101), puis sauvegarder
  • Et voila l'utilisateur est créé, si je souhaite modifier les identifiants et mdp de la ligne (ce qui nous permettra de nous connecter aux softphones comme 3CX et Zoiper), je doit copier le fichier IPBX.ini a partir de ma VM serveur dans un répertoire temporaire:
    • docker cp xivo_webi_1:/etc/xivo/web-interface/ipbx.ini /tmp
  • ensuite utiliser la commande nano /tmp/ipbx.ini et modifier dans la catégorie [user] readonly-idpwd en false puis sauvegarder
    • Enfin, remettre le fichier modifié avec la commande docker cp /tmp/ipbx.ini xivo_webi_1:/etc/xivo/web-interface/ipbx.ini
  • redémarrer Asterisk puis recharger la page, enfin aller voir dans les paramètres des lignes si c'est fonctionnel.
  • Autre chose importante: si on veut que l'utilisateur utilise des numéros spécifiques, il faut choisir le contexte contenant ces numéros dans lignes (si il est grisé, déliez la ligne de l'utilisateur concerné dans Utilisateurs en enlevant le numéro dans l'onglet lignes)
