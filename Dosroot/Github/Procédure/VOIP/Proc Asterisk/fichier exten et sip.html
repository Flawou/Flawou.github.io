fichier exten et sip

Copier le fichier de configuration pour ne pas le perdre et pouvoir le remettre en cas de soucis

cp /etc/asterisk/sip.conf /etc/asterisk/sip.conf.original

ensuite vider le fichier de base 

.> /etc/asterisk/sip.conf

modifier le fichier 

nano /etc/asterisk/sip.conf

exemple configuration

ensuite modifier le fichier contenant les extensions, le sauvegarder avant modif.

cp /etc/asterisk/extensions.conf /etc/asterisk/extensions.conf.original

puis > /etc/asterisk/extensions.conf 

le modifier 

nano /etc/asterisk/extensions.conf

puis vérifier que les utilistaeur soit bien présent 

asterisk -rvv

reload 

sip show users
