Messagerie Vocale

Nous devons commencer par modifier le fichier extensions

nano /etc/asterisk/extensions.conf

exten => 098, 1, VoiceMailMain (${CALLERID(num)}) 

explication :

extensions → Numéro du téléphone , priorité 1, MessagerieVocal (identifiant de la personne qui appel + son numéro)

Maintenant nous allons configuer le fichier de la boîte message : 

cp /etc/asterisk/voicemail.conf /etc/asterisk/voicemail.conf.original

on vide le fichier de base 

→ /etc/asterisk/voicemail.conf

puis éditer le fichier 

nano /etc/asterisk/voicemail.conf

explication :

100 = numéro de téléphone utilisateur 123 = mot de passe de la mesagerieV, puis adresse mail

Reload les paramètres dans la console Asterisk 

asterisk -rvv ; ouvrir la console

reload ; recharger pour prendre en compte les changements
