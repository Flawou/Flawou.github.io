Connexion domain

 

Pour se connecter au domaine via le client, entrer dans une session user classique pour récuperer les informations réseaux via le DHCP 

(vérifier que le DNS est équivalent à l'ip du serveur)

ensuite, clique droit sur “Ce PC” puis renommer le pc (avancé) 

Puis cliquer sur modifier et finir par choisir le nom de domaine 

ensuite, changer d'utilisateur en utilisent la stratégie de groupe ( accessible en écrivant gpedit.msc dans w+r)

Puis sélectionner Paramètres windows dans configuration ordinateur, paramètres de sécurité 

ensuite, ouvrir , stratégie locale, option de sécurité puis clique droit sur Ouverture de session interactive : Ne pas afficher le nom du dernier utilisateur connecté puis propriété, activer.
