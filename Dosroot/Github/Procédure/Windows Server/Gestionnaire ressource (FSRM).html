Gestionnaire ressource (FSRM)

Pour activer le gestionnaire de ressource,

Ouvrir le gestionaire de serveur 

Cliquer sur ajouter des rôles, suivant jusqu'au choix du rôle, cliquer sur le menu déroulant ( petite flèche sur la gauche) deux fois puis cocher le Gestionnaire.

Une fois l'installation terminée, ouvrir le Gestionnaire de ressources du serveur

Une fois le gestionnaire ouvert, nous allons créer un modèle de quota, pour cela, aller sur Gestion de Quota, ensuite clique sur modèles de quotas , puis créer un modèle de quota en faisant un clique droit dans la fenêtre centrale.

 Configurer le nouveau quota en indiquant le nom du modèle 1 , la limite 2 , choisir le type (inconditionnel ou conditionnel) 3 et cliquer sur OK 4 pour le créer.

Ensuite, pour appliquer le quota sur un dossier voulu, cliquer sur gestion de quota, puis Quotas et  créer un quota en faisant un clique droit dans la fenêtre centrale.

Choisir le dossier voulu via parcourir ou en écrivant directement l'UMC, puis choisir la limite désiré pour ce dossier.
