Install + Config DHCP

Ouvrir le gestionaire de serveur 

Cliquer sur ajouter des rôles

Cliquez sur suivant jusqu'à atteindre le choix du serveur et choisir son rôle ( ici DHCP ) puis suivant jusqu'à terminer la configuration.

Puis cliquer sur DHCP et clique droit sur le serveur > gestionaire DHCP

Vous trouverez toutes les options pour configurer ce rôle :

Nous allons créer notre première étendue, pour cela, clic droit IPv4 sous le nom du serveur, puis “Nouvelle étendue” , puis choisir la plage ip en fonction des besoins, ne pas oublier la passarelle dans le compte 

Avancer puis choisir la durée du bail en fonction des besoins puis, en poursuivant l’assistant, vous êtes guidé dans les options les plus utiles à paramétrer :

  • routeur (passerelle par défaut) ;
  • nom de domaine et serveurs DNS ; 
  • serveurs WINS.

Ici nous allons simplement mettre le routeur. 

Pour finir, activer l'étendue.

Afin de fonctionner correctement, il reste une dernière étape à mettre en œuvre : autoriser les flux au niveau du pare-feu Windows.  Aller dans les outils d’administration (dans le menu Démarrer) et ouvrez le “Pare-feu Windows avec fonctions avancées de sécurité”. Créez une “Règle de trafic entrant” en spécifiant le protocole UDP sur les ports 67 et 68, puis autorisez ce flux et nommez la règle DHCP par exemple.
