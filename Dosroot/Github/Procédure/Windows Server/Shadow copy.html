Shadow copy

Pour activer une shadow copy sur un disque, commencer par ouvrir le gestionnaire de disque (w+x, gestionnaire de disque)

Ensuite, sélectionner le disque qui stock les fichiers nécessaire, clique droit propriétés

Aller dans clichés instantanés, puis activer 

Ensuite, mettre les paramètres désirers dans la section Paramètres

Ajuster la taille maximum et la planification (dans Planifier)

 

il est possible de créer des sauvegardes instantanée en cliquant sur  créer
