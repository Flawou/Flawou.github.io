Modifier la stratégie de mot de passe

Pour modifier la stratégie de mot de passe localement, 

Ouvrir le gestionaire de serveur  puis outils, Centre d'administration Active Directory

 

 

ensuite aller dans le fichier system du serveur

Puis écrire pass dans le filtre de recherche, entrer dans le fichier Password Settings Container

puis cliquer sur nouveau, paramètres de mot de passe 

ensuite, choisir les paramètres, nom de la stratégie à qui elle s'applique etc

Pour supprimer une règle, clique droit sur la règle, propriétés puis décocher la protection contre la suppression accidentelle.
