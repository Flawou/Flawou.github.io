Profil itinérant

Sert à sauvegarder le profil utilisateur sur un dossier partagé sur serveur ( paramètrages / dossier ) exemple : charge le profil et les données à la connexion et les synchronise (sauvegarde) à la déconnexion
Avant de configurer un profil itinérant, nous devons créer un dossier répertoire pour stocker les paramètres et leurs fichiers.

Partagé le fichier aux utilisateurs de l'AD et supprimer les perms pour “Tout le monde”, remplacé par utilisateurs

Ajusté les droits NTFS (sécurité) pour ce groupe en contrôle total

Prendre le chemin UNC et le mettre sur l'utilisateur désiré : Clique droit propriétés > Profil > Chemin du profil
