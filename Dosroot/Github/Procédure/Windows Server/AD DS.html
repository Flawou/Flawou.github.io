AD DS

En premier lieu : Changer le nom, mise à jour off,  adresse ip fixe + découverte réseau,

Ouvrir le gestionaire de serveur 

Cliquer sur ajouter des rôles

Cliquer sur suivant jusqu'à atteindre le choix du service et choisir son rôle (ici AD DS) puis suivant jusqu'à l'installer.

Cliquer sur la drapeau pour promouvoir le serveur en contrôleur de domaine

Ajouter une nouvelle forêt

suivant *x puis installer, redémarrage du serveur.

Cliquer sur outil, domaines et approbations Active Directory

 

clique droit sur le serveur puis géré 

 

clique droit sur le serveur > nouveau > Unité d'organisation (Autant que vous en avez besoin)

Puis créer les utilisateurs dans l'unité d'organisations désirés

Pour supprimer un OU, il faut commencer par activer l'affichage des fonctionnalités avancées

ensuite, clique droit propriété sur l'OU que l'on souhaite supprimer.

Sélectionner Objet et décocher la case de Protection.

Vous pouvez désormais supprimer l'OU.
