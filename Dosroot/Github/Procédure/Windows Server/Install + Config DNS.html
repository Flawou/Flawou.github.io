Install + Config DNS

Ouvrir le gestionaire de serveur 

Cliquer sur ajouter des rôles

Cliquer sur suivant jusqu'à atteindre le choix du serveur et choisir son rôle (ici Server DNS)

Puis cliquer sur DNS et cliquer droit sur le serveur > gestionaire DNS.

Nous allons créer notre première zone, pour cela sélectionner Zone de recherche directe, puis action et nouvelle zone. 

Puis suivant et laisser la sélection de base Zone principale puis suivant.

Nommer la zone à votre bon vouloir. Puis suivant, suivant et pour finir, terminer.

Ensuite, cliquer sur le fichier et Nouvel hôte

 

Nommer l'hôte et associé le server dédié pour celui-ci.

Pour avoir une réponse DNS, il faut ping le nom de l'hôte + le nom de route: exemple : locale.test

Puis acitver la zone de recherche inversée

Nouvelle zone →

 

Suivant, nommé l'ip du serveur puis terminer
