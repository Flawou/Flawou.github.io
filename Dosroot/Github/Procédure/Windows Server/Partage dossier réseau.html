Partage dossier réseau

1.Rendre le dossier accessible depuis le serveur
Nous allons voir comment partager un dossier sur notre réseau

Pour commencer, choisir le dossier que l'on souhaite partager, faire un clique droit dessus et cliquer sur propriétés.

Ensuite, aller dans la section partage

puis partage avancé

cocher partager ce dossier puis OK.

Dans la même section, cliquer cette fois-ci sur Partager

Puis, choisir le groupe d'utilisateur ou l'utilisateur concerné, ici, ce sera pour tout le monde.

Puis cliquer sur Ajouter. Ajuster le niveau d'autorisation que vous souhaitez, ici nous laissons lecture.
Ensuite cliquer sur Partager (en bas à droite)

Sélectionner Oui, puis terminer.
