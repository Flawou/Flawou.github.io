Activer les partages de fichiers

 

Pour pouvoir activer le partage de fichier sur le serveur, nous commençons par ouvrir le gestionnaire de serveur, pour cela : 

  • Appuyer sur la touche windows
  • Taper gestionnaire de serveur

Ensuite, sélectionner Serveur local.

Une fois dans serveur local, cliquer sur Public : actif, en face de Pare-feu Windows Defender

Cliquer sur : Autoriser une application via le pare-feu 

Cocher la fonction : Partage de fichiers et d'imprimantes.
 

N'oubliez pas de valider en appuyant sur Ok, vous avez désormais terminé!
