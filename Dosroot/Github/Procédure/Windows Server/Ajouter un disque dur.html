Ajouter un disque dur

Pour ajouter un disque dur sur le serveur, nous allons d'abord ajouter le disque ( ou une partie d'un disque )

Pour celà, clique droit sur le serveur concerné puis settings.

Une fois la page ouverte, nous allons cliquer sur “add…” en bas de la page.

Nous sélectionnons Hard disk, avec l'interface de connexion que nous désirons pour la machine, ici, nous allons choisir SATA.

  • Allouer l'espace désiré

Puis choisir l'endroit où nous allons stocker les données de ce disque, ou le disque voulu directement.

Une fois cette étape réaliser, nous allons rendre le disque lisible pour le server, pour cela :

sélectionner le type de partition voulu pour le disque, ici “Partition GPT” puis OK.

 

Le disque est maintenant visible.
