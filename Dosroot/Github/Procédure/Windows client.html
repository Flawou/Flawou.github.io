Windows client

Etape 1 : Renommer votre machine

Pour renommer votre machine, clique gauche sur l'icone windows, puis Paramètres.

Sélectionner Système, faire défiler la colonne jusqu'à voir apparaître À propos de puis cliquer dessus.

Cliquer sur Renommer ce PC et le nommé à votre bon vouloir.

 

Etape 2 : Suspendre les MàJ

Maintenant, nous allons suspendre les mises à jours pour une durée de 7 jours.

Retourner dans le menu paramètres comme à l'étape 1, puis sélectionner mise à jour et sécurité.

Puis cliquer sur Suspendre les mises à jour pendant 7 jours.

 

Etape 3: Définir une ip sur la machine

Pour finir, nous allons fixer notre ip :

Appuyer simultanément sur la touche windows + R 

Taper ncpa.cpl

Cliquez droit sur votre interface réseau puis propriétés

Puis sélectionner Protocole internet version 4 (flèche verticale), puis sélectionner propriétés (flèches horizontale)

 

Une fois l'interface ouverte, cliquer sur Utiliser l'adresse IP suivante, et insérer l'ip voulu.

 

Le dns est facultatif, à votre bon vouloir.
