Liasion OCS GLPI

Commencer par activer les plugins sur glpi

ensuite installer git sur la machine :

Apt install git 

ce mettre dans fichier glpi/plugins 

exemple : /var/www/glpi/plugins ou /var/www/html/lpi/plugins

ensuite aller chercher le wget sur git hub

exemple : wget https://github.com/pluginsGLPI/ocsinventoryng/releases/download/2.0.4/glpi-ocsinventoryng-2.0.4.tar.bz2

puis décompresser l'archive :

tar -xzf glpi tab

retourner sur glpi pour activer le nouveau plugin (refresh pour le voir apparaître)

une fois le plugin actif, aller dans Outil > OCS INVENTORY NG 

Import de l'inventaire > cliquer sur le + Importer ou lier des ordinateurs 

config : 

Puis activer le TRACER sur OCS 

Configuration → Configuration Général → Serveur → On Trace_DELETED
